# Forecasting con Indicadores Técnicos

Proyecto académico de forecasting de corto plazo utilizando indicadores técnicos para generar señales automáticas de trading.

## Objetivo
Desarrollar un modelo sistemático que genere señales BUY / SELL / HOLD usando RSI, MACD y medias móviles.

## Activos analizados
AAPL, TSLA, META, GOOGL, NVDA, MSFT, AMZN

## Estructura del repositorio
- data: datos históricos
- notebooks: análisis exploratorio
- src: implementación del modelo
- results: señales y métricas
- docs: documentación y entregables

## Tecnologías
Python, pandas, yfinance, TradingView, Alpaca
